# Домашнее задание 1

**Как запустить проект:**
1. Активировать виртуальное окружение: `source venv/bin/activate`
2. Установить зависимости: `pip install django`
3. Запустить сервер: `python manage.py runserver`

**Доступные страницы:**
* Главная: `/index/`
* Вопрос: `/question/1/`
* Спросить: `/ask/`
* Логин: `/login/`
* Регистрация: `/register/`
* Настройки: `/setting/`

**Технические детали (Шаблонизация):**
В проекте используется наследование шаблонов Django. 
Создан главный каркас `base.html`, который содержит шапку сайта, правую колонку.
